# Отчёт о проделанной работе
**Период:** 14 неделя второго сезона проекта  
**Дата составления:** 05.06.2026  

## Описание проделанной работы

За четырнадцатую неделю работы над проектом были выполнены следующие шаги по переходу на полноценное взаимодействие с бекендом:
* Полный переход от старых запросов к актуальным из swagger
* Реализована многоуровневая синхронизация данных (Предметы -> Группы -> Студенты)
* Исправлена критическая ошибка "No token found", приводившая к разлогину
* Добавлена поддержка Bearer-авторизации для всех запросов к серверу
* Реализовано локальное хранение занятий в Room для отображения реальной истории(вместо мок)

## Поэтапное описание

### Синхронизация данных
Реализован алгоритм для сбора данных: приложение получает список доступных преподавателю предметов, обходит каждую группу и выкачивает актуальный список студентов, сохраняя их в локальную room БД

### Безопасность и авторизация
  * Исправлен жизненный цикл токена: теперь `auth_token` не удаляется при очистке локального кэша студентов(room)
  * Все сетевые запросы приведены к стандарту JWT (добавлен префикс Bearer)
  * Добавлена обработка различных форматов ID пользователя (user_ID/user_id) для корректного парсинга профиля

### История занятий
  * В Room добавлена entity `Lesson` для хранения истории созданных сессий
  * Теперь вкладка "История" у преподавателя отображает реальные данные о прошедших парах, а не моковые данные
