# Отчёт о проделанной работе
**Период:** 18 неделя проекта  
**Дата составления:** 23.01.2026  

## Описание проделанной работы

За восемнадцатую неделю разработки были успешно выполнены следующие шаги:
* Дописана часть программы отвечающая за занятия
* Найдены и исправлены баги с shared prefs + NFC
* Дописана серверная часть кода для поддержки новых операций
* Реализована авто генерация журнала посещения при завершении занятия
* Немного изменён интерфейс

## Поэтапное описание (основные изменения)

 В ходе работы был дописан функционал занятий. Теперь возможно начать занятие и сразу же отмечать студентов с отображением их в списке
 После того как admin нажимает на кнопку завершения реализуется следующая логика:
 * Т.к в один момент времени у группы может быть одно занятие студенты отмечаются на 1 занятии
 * В поле attendance DB ставится true
 * По завершении занятия автоматически формируется html отчёт присутствия на сервере
 * Затем всем студентам из группы ставится false в поле attendance
